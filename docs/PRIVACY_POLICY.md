# Privacy Policy

**Last Updated: March 8, 2026**

## 1. Introduction

This Privacy Policy describes how Miku ("we", "our", or "the bot") collects, uses, and protects information from users of our Discord bot and web dashboard.

## 2. Information We Collect

### 2.1 Automatically Collected Data

When you use Miku, we automatically collect:

**Discord User Data:**
- User ID (Discord unique identifier)
- Username and discriminator
- Avatar URL
- Server (Guild) ID

**Activity Data:**
- Message count per server
- XP (experience points) earned
- Level progression
- Timestamps of activity

**Server Data:**
- Server (Guild) ID
- Server name and icon
- Channel IDs (for level-up announcements)
- Role IDs (for role rewards)
- Server member count

### 2.2 Dashboard Data

When you use the Miku Dashboard:
- Discord OAuth2 session data
- IP address (for security purposes)
- Browser type and version
- Access timestamps

## 3. How We Use Your Information

### 3.1 Core Functionality
- Track user progress and levels within Discord servers
- Display leaderboards and statistics
- Assign role rewards based on level achievements
- Send level-up notifications

### 3.2 Dashboard Functionality
- Authenticate server administrators
- Display server statistics and leaderboards
- Allow configuration of bot settings

### 3.3 Service Improvement
- Monitor bot performance and reliability
- Identify and fix bugs
- Improve user experience

## 4. Data Storage and Security

### 4.1 Storage
- Data is stored in a secure PostgreSQL database
- We use industry-standard encryption for data transmission
- Database credentials are stored securely and not publicly accessible

### 4.2 Security Measures
- OAuth2 authentication for dashboard access
- Regular security updates
- Access logs for monitoring
- Secure API endpoints with authentication

### 4.3 Data Retention
- User data is retained as long as the bot is active in a server
- Data may be cached temporarily for performance
- Upon bot removal from a server, data may be retained for up to 30 days

## 5. Data Sharing

### 5.1 We Do NOT:
- Sell your personal information
- Share data with third-party advertisers
- Use data for marketing purposes
- Share data outside of necessary service operations

### 5.2 We May Share:
- Publicly displayed leaderboard information (usernames, levels, XP)
- Aggregated, anonymous statistics
- Data when required by law

## 6. Third-Party Services

Miku uses the following third-party services:

### Discord API
- Used for bot functionality
- Subject to [Discord's Privacy Policy](https://discord.com/privacy)

### Hosting Services
- Render.com (API hosting)
- Vercel (Dashboard hosting)
- Neon.tech (Database hosting)

These services have their own privacy policies that govern their data practices.

## 7. Your Rights

### 7.1 Data Access
You can view your data through:
- Dashboard leaderboards
- Bot commands (`/rank`, `/xp`)

### 7.2 Data Deletion
You can request data deletion by:
- Removing the bot from your server (server administrators)
- Contacting us directly for personal data removal

### 7.3 Server Administrators
Server administrators can:
- View all server member data through the dashboard
- Reset user levels using bot commands
- Remove the bot to stop data collection

## 8. Children's Privacy

Miku complies with Discord's Terms of Service, which requires users to be 13 years or older. We do not knowingly collect information from users under 13.

## 9. Cookies and Tracking

### Dashboard Cookies
The Miku Dashboard uses:
- Session cookies for authentication
- Functional cookies for user preferences
- No advertising or tracking cookies

## 10. Data Breach Notification
